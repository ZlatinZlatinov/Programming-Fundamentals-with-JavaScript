# Lecture 3: Arrays 

## Content: 

#### Definition: 
 - Sequence of elements
 - Can store multiple values in one variable 
 - Elements are numbered from 0 to array's length -1 

#### Methods: 
 - Arrays are special objects 
 - they have buit-in properties and methods (like .length) 

#### Array iteration: 
 - Using standard loop 
 - Using 'for - of' loop
